== Description ==

Shipping By Ranges is a plugin for TheCartPress, eCommerce plugin for WordPress. It allows to calculate the shipping cost by product.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload Shipping By Ranges for TheCartPress to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
